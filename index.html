Sampler

ブラウザだけで動く、オフライン対応の音楽サンプラー。
スマートフォンのホーム画面に追加すれば、ネイティブアプリのように使えます。

アプリを起動する →

初回起動時はオンラインでアクセスしてください。以後はオフラインで起動できます。

📲 ホーム画面に追加 (PWA インストール)

iPhone (Safari)

  1. このページを Safari で開きます (Chrome では不可)。
  2. 画面下部 (または右上) の 共有ボタン ⬆︎ をタップ。
  3. メニューから 「ホーム画面に追加」 を選びます。
  4. 右上の 「追加」 をタップ。
  5. ホーム画面の Sampler アイコンから起動できます。

※ iOS の仕様上、初回起動時にマイクの利用許可が求められます。

Android (Chrome)

  1. このページを Chrome で開きます。
  2. 右上の ⋮ メニュー をタップ。
  3. 「アプリをインストール」 または 「ホーム画面に追加」 を選択。
  4. 表示される確認ダイアログで 「インストール」。
  5. アプリ一覧の Sampler から起動できます。

※ 一部の Android ブラウザでは画面下部にインストールバナーが表示されます。

PC (Chrome / Edge)

  1. アドレスバー右端の インストールアイコン をクリック。
  2. または ⋮ メニュー → 「Sampler をインストール」。

✨ 主な機能

  • 🎙 録音 / オーディオファイル取り込み
  • 🎹 4×4 パッド + クロマチック・ピッチ演奏
  • ✂ トリム / スライス / ループ
  • 🌊 リバーブ / フィルター / ディレイ
  • 🥁 16 ステップ・シーケンサー (BPM / スイング)
  • 🔊 ノーマライズ
  • 🪓 自動チョップ (オンセット検出)
  • ♻ リサンプリング (FX 込み)
  • ⏱ タイムストレッチ (ピッチ独立)
  • ⏪ スキップバック (30 秒遡り録音)
  • 👆 ベロシティ感度
  • 🎛 Web MIDI 対応
  • 📤 WAV エクスポート / プロジェクト保存
  • 📚 チュートリアルモード
  • 🐦 動物の鳴き声 / 雨 / 波 / 雷など環境音も標準同梱

🎛 操作の流れ

  1. 1. 音を選ぶ / 録音する

    左の「ライブラリ」から内蔵音源を選び、パッドへドラッグ。
    または 🎙 録音ボタンでマイクから録音、📁 取り込みでファイルを読み込み。

  2. 2. 演奏する

    パッドをタップ / クリック。強さで音量とニュアンスが変化します (ベロシティ)。
    キーボードでは 1〜4, Q〜R, A〜F, Z〜V がパッドに対応。

  3. 3. 加工する

    パッドを長押しで編集モードに。波形上で範囲選択 → トリム / ノーマライズ / スライス。
    FX パネルでリバーブ・フィルター・ディレイ・タイムストレッチ。

  4. 4. シーケンスを組む

    下部の16 ステップを点灯させてリズムを打ち込み。BPM・スイングは右上で調整。
    ▶︎ で再生、●REC でリアルタイム録音。

  5. 5. 書き出す

    📥 エクスポートから WAV を保存、または共有から SNS へ。
    プロジェクト全体は 💾 保存 でローカル保存できます。

💡 困ったときは

  • 音が出ない場合は、画面を一度タップしてください (iOS の制約により初回タップで音声出力が解放されます)。
  • マイクが使えない場合は、ブラウザの設定でマイク権限を許可してください。
  • オフライン化が完了するまで初回は通信が必要です。完了後は機内モードでも動作します。
